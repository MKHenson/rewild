# RE-WILD

Rewild is a game about time travel and natural history. Built with WASM and WebGPU

Checout the [demo here](https://mkhenson.github.io/rewild/)

### How do I get set up?

Clone the project and install the dependencies

```
npm intall
```

Build the project

```
npm run build
```

Run the prroject

```
npm run start
```
